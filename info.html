Enkel konst

Denna hemsidan fungerar på så sätt att den hämtar 12st konstverk från "Art institute of Chicago", detta gör den via deras egna API "https://api.artic.edu/api/v1/artworks". Sedan presenterar hemsidan dessa 12 konstverk på så sätt att man kan scrolla igenom alla, eller alternativt använda sök- funktionen för kolla om hemsidan har ett specifikt konstverk.
